What is it?

FILE!IT is a use-anywhere web application to browse, share and view files in an innovative new way. It leverages the best of current and upcoming web technologies, and is bound to standards.

Users interact with the application in a familiar domain–the web browser–which means it can be accessed from anywhere. The application features intuitive controls based on industry standards and human-interface guidelines.

The application aims to allow users access to their files stored anywhere. The user will be capable of configuring access points (providing they have the necessary privileges) and view file hierarchies for their storage servers.

Some front-end enhanced features include inline viewing of video, audio, images and text files.
